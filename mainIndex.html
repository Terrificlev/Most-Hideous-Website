be used with one another. One example may be:
Brown and Orange.

neon green and neon pink also.

Overlapping

Overlapping is very bad because it shows that you want to cramp a lot of information into 1 small place which is bad because it will draw a lot of attention which if there is another element wanting its attention then it will be a bad experience for the user to try to understand how the website works.
